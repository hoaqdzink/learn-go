## Phase 3 — Deep Go Internals

Phase này giúp bạn hiểu **Go chạy như thế nào bên trong**, nền tảng để tối ưu performance và debug các bug khó.

### Nội dung chính:

- `memory-model/` — stack vs heap, escape analysis
- `garbage-collector/` — cách GC hoạt động, mark-sweep, GC tuning cơ bản
- `scheduler/` — mô hình G-M-P, goroutine scheduler
- `runtime-internals/` — lifecycle goroutine, stack growth, memory reuse

Khuyến nghị: luôn kèm theo **benchmark / profile nhỏ** khi học phần này.
