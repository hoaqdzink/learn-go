## Phase 4 — Backend Engineering với Go

Mục tiêu: từ Go core → **build được backend service production-ready**.

### Nội dung chính:

- `net-http/` — viết REST API thuần `net/http` (không framework)
- `framework/` — chọn 1 (Gin hoặc Echo) và build lại API
- `rest-api/` — status code, middleware, validation, DTO
- `clean-architecture/` — `handler/`, `service/`, `repository/`, `model/`
- `authentication/` — JWT, refresh token, OAuth2

Nên có 1 mini project xuyên suốt phase này (ví dụ: Todo API / Auth API).
