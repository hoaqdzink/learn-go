## Project Roadmap

Mục tiêu của thư mục `projects/` là biến toàn bộ kiến thức trong các phase thành **sản phẩm thực tế**.

### Các level:

- `level-1/` — CLI nhỏ, xử lý file
  - `cli-todo-app/`
  - `file-parser/`
- `level-2/`
  - `crud-rest-api/`
- `level-3/`
  - `auth-service/`
- `level-4/`
  - `payment-service/`
- `level-5/`
  - `order-microservice/`
- `level-6/`
  - `kafka-event-system/`
- `level-7/`
  - `distributed-scheduler/`

Mỗi project nên có:

- `README.md` mô tả yêu cầu/feature
- `design.md` (nếu cần) mô tả kiến trúc
- source code Go tương ứng
