## Phase 10 — Senior Engineering Mindset

Mục tiêu: từ “biết làm” → **biết thiết kế, tối ưu, review, đưa quyết định kỹ thuật**.

### Nội dung chính:

- `code-reading/` — đọc & hiểu source người khác
- `debugging-production/` — debug bug production, log, tracing
- `query-optimization/` — tối ưu SQL, index, plan
- `architecture-review/` — review thiết kế hệ thống
- `tradeoff-estimation/` — đánh giá trade-off, estimate effort

Phase này nên song song với việc **đọc code open-source + làm project thật**.
