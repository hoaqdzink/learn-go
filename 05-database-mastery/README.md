## Phase 5 — Database Mastery

Mục tiêu: **SQL và PostgreSQL rất chắc**, dùng Go để làm việc với DB hiệu quả.

### Nội dung chính:

- `sql/` — join, subquery, CTE, index, execution plan
- `postgres/` — MVCC, transaction isolation, deadlock, vacuum
- `go-db-libs/` — `pgx`, `sqlx`, sau đó `gorm`
- `redis/` — cache-aside, TTL, distributed lock

Phase này nên kèm rất nhiều **query thực tế + tối ưu index**.
