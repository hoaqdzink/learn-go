## Phase 2 — Core Go

Mục tiêu: **nắm chắc cú pháp Go và các khái niệm cốt lõi** để có thể viết chương trình Go rõ ràng, idiomatic.

### Nội dung chính:

- `syntax/` — cú pháp cơ bản, kiểu dữ liệu, khai báo biến/hằng
- `functions/` — function nâng cao: variadic, multiple return, closure, anonymous function
- `struct-method/` — struct, method, value receiver vs pointer receiver
- `pointer/` — con trỏ, `*` và `&`, truyền tham chiếu
- `interface/` — interface, duck typing, design theo interface
- `error-handling/` — `error`, pattern `if err != nil`, wrap error

Mỗi thư mục con sẽ có **ghi chú + ví dụ + bài tập nhỏ**.
