## Phase 1 — Foundation

Phase này tập trung vào **CS Core, Hệ điều hành và Networking**, là nền tảng để sau này hiểu sâu về concurrency, distributed systems và performance.

Trước khi vào Phase 1, nên hoàn thành **`00-go-setup/` (cài Go & làm quen công cụ)** để không bị vướng phần môi trường.

### Bạn nên bắt đầu từ:

1. `cs/data-structures`, `cs/algorithms`, `cs/complexity`
2. `os/process-thread`, `os/memory`, `os/sync-deadlock`
3. `networking/tcp-ip`, `networking/http-https`, `networking/dns-socket`

Mỗi thư mục con sẽ có:

- Ghi chú ngắn (file `README.md`)
- Ví dụ code (Go hoặc pseudo khi cần)
- Bài tập nhỏ / câu hỏi tự trả lời
