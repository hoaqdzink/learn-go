## Algorithms — Dynamic Programming (DP)

### 1. Mục tiêu

- Hiểu 3 ý tưởng cốt lõi:
  - Overlapping subproblems.
  - Optimal substructure.
  - Memoization vs Tabulation.
- Tự giải được vài bài DP cơ bản.

### 2. Nội dung chính

- Fibonacci tối ưu bằng:
  - Memoization (top-down).
  - Tabulation (bottom-up).
- Bài climbing stairs.
- (Tuỳ bạn mở rộng thêm bài coin change, LIS, v.v.)

### 3. Bài tập gợi ý

- Cài 3 version Fibonacci:
  - Recursion naive.
  - Recursion + memoization.
  - Bottom-up.
- Bài climbing stairs:
  - Mỗi lần bước 1 hoặc 2 bậc → đếm số cách lên bậc `n`.
