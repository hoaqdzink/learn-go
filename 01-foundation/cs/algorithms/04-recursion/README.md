## Algorithms — Recursion

### 1. Mục tiêu

- Hiểu rõ:
  - Base case.
  - Recursive case.
- Thấy được mối liên hệ recursion ↔ call stack ↔ stack overflow.

### 2. Nội dung chính

- Các ví dụ kinh điển:
  - `factorial`.
  - `fibonacci` (naive).
  - Duyệt tree bằng recursion.
- Cách “dịch” recursion sang vòng lặp (khi cần tối ưu).

### 3. Bài tập gợi ý

- Viết:
  - `Factorial(n int) int`
  - `Fib(n int) int` (naive).
- Viết hàm dùng recursion để:
  - Đếm số node trong tree.
  - Tính chiều cao tree.
